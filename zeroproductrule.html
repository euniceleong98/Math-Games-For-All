Zero Product Rule Challenge

Use the rule: if a product is zero, at least one factor must be zero. Solve each equation for x!

Choose your level

Each game has 5 fresh questions. The questions are randomly created every attempt, so students will not see the same quiz again and again.

Easy

Simple factors

All coefficients of x are 1, for example:
(x − 1)(x + 2) = 0

Medium

Mixed coefficients

Some coefficients of x are not 1, for example:
(2x − 1)(−3x + 1) = 0

Hard

Expanded quadratics

The equation is shown in expanded form, but it is factorisable using the Zero Product Rule.

Teacher tip: The timer pauses whenever feedback is shown, so students are not penalised for reading the explanation.
